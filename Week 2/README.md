# SOC Project – Week 2

## Overview
This project demonstrates SOC operations using Wazuh SIEM.

## Tools Used
- Wazuh
- Metasploit
- VirusTotal
- AlienVault OTX
- Linux CLI

## Steps

1. Generated alerts using SSH brute-force
2. Classified alerts using MITRE (T1110)
3. Performed triage using VirusTotal & OTX
4. Created incident response report
5. Preserved evidence using SHA256 hashing
6. Simulated attack using Metasploit
7. Detected and blocked attacker activity

## Outcome
Completed full SOC workflow including detection, analysis, response, and reporting.
